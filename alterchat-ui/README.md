# AlterChat UI

Tauri v2 + React + Vite + TypeScript tabanlı masaüstü uygulaması.

Proje felsefesi, mimari ve güvenlik detayları için: [Ana README](../README.md)

## Geliştirme

```bash
npm install
npm run tauri dev
```

## Gereksinimler

- Rust (stable)
- Node.js v18+
- Tauri v2 sistem bağımlılıkları: https://v2.tauri.app/start/prerequisites/

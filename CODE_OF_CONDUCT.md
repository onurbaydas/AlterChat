# Code of Conduct

AlterChat is a privacy and security project. Good technical work here depends on
calm, respectful collaboration, especially when discussions involve risk,
identity, cryptography, or personal safety.

## Our Standards

Expected behavior:

- be respectful and specific
- assume good faith while still reviewing carefully
- criticize code and ideas, not people
- explain security concerns with evidence
- make space for newcomers to ask questions
- keep discussions focused on the project
- handle sensitive reports privately

Unacceptable behavior:

- harassment, threats, or personal attacks
- discriminatory language or imagery
- publishing private information without consent
- pressuring maintainers to disclose vulnerabilities early
- trolling, sustained disruption, or bad-faith review behavior
- sexualized comments or unwelcome attention

## Security Discussions

If a discussion includes exploit details, private keys, message contents,
database dumps, or a practical attack path, move it to the private reporting
process in [SECURITY.md](SECURITY.md).

## Enforcement

Maintainers may:

- edit or remove comments
- close issues or pull requests
- request a change in tone or process
- temporarily or permanently restrict participation

Enforcement decisions should be proportional, documented when appropriate, and
focused on keeping the project safe and workable.

## Reporting Conduct Issues

Contact the maintainers privately. Include links, screenshots, or context when
available. Reports will be handled with discretion.
